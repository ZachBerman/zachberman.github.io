About






The way we as humans communicate has changed so drastically over the last 10 years, and continues to do so. Watching this evolution is quite amazing. The internet has made major changes to ways the human communicate, and has made connecting people from different sides of the world an easy task. Web development, and how we communicate using the internet is one of my passions. Studying communications in college, the way all of the pieces fit together is extremely interesting to me. My methodology on the internet is based around the efficiency of communication, and how the user interacts with the page. New technologies are emerging all the time giving us new ways make connections, find services, and trade information. I believe that websites and applications should be clean, fast, efficient, and accessible on all devices.

On the Web


Backcountry Wisdom

Backcountry Wisdom is a simple quotes database and generator that allows the user to contribute quotes that will be displayed as they are shuffled through. As an avalanche educator this is a neat resource of one-line quotes for novice backcountry travelers to click through.


Technologies used: Ruby on Rails, HTML5, CSS3, Bootstrap, Github, and Heroku.


BozeGuide

BozeGuide is Yelp clone that integrates the Google Map API, along with user authenication, image uploading, comments, and ratings. This project was started as a guide to Bozeman, MT area breweries and restuarants.


Technologies used: Ruby on Rails, Google Map API, HTML5, CSS3, Bootstrap, Github, and Heroku.


Vidshare

Vidshare is a two-sided video streaming educational platform. Teachers can upload videos, pictures and lessons using an advanced Javascript enabled UI. User authentication, credit card processing, and video/image uploading also supported.


Technologies used: Ruby on Rails, Javscript/Jquery, HTML5, CSS3, Bootstrap, Github, and Heroku.


Grammable

Grammable is an Instagram clone that allows for image uploading and user authentication/comments. This app was built using Test Driven Development ensuring well-tested code was deployed throughout the process.


Technologies used: Ruby on Rails, Test Driven Development, HTML5, CSS3, Bootstrap, Github, and Heroku.


Lister

Lister is a simple single page to-do list app. Using Javacript this app allows for a fluid user interface without the need to refresh the page.


Technologies used: Ruby on Rails, Test Driven Development, Javascript, HTML5, CSS3, Github, and Heroku.
